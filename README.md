# My Idena

Idena iOS and Android app based on Flutter framework .

## Configuration

In the first launch, go to "Param" menu
Tape "http://10.0.2.2:9119" to connect to your local node 
and tape the api.key (cf %appdata%\Idena\node\datadir\api.key)
